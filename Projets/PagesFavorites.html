PAGES DE FAVORIS

Concept du Projet

Ce projet est une page Web qui répertorie les sites favoris d'un utilisateur que celui-ci rajoute manuellement grâce à un formulaire.
J'ai réalisé ce projet pour mon utilisation personnelle.

Début du Projet

Au début du projet, j'ai identifié les technologies utiles à la réalisation de cette page web : HTML, CSS, PHP et éventuellement un framework JavaScript .
